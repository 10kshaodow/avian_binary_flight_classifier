# avian_binary_flight_classifier
**Bird Flight Image Classifier** — This project builds a binary image classifier that predicts whether a bird is airborne or not airborne from still images. It uses transfer learning in PyTorch, dataset preprocessing and splitting, and a YOLO-based bird detection stage to crop images around the bird before classification. The goal is to improve robustness by reducing reliance on scene context such as sky, branches, or water and making the model attend more directly to the bird’s pose and appearance.
